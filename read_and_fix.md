# How to Improve Poor Translated
1. Clean the Source Text
- Remove all non-Burmese characters and fix encoding issues before translation.

2. Use Better Translation Settings
- Use a higher-quality model (qwen2.5:14b, qwen:7b)
- Add these instruction to your prompt:
	- Translate to natural, literary Burmese prose
	- Avoid literal word-for-word translation
	- Use varied vocabulary (don't repeat the same phrases)
	- Ensure character names are consisten
	- Maintain narrative flow like a novel
3. Post-Editing Steps
- Remove  repetitive sections
- Verify character names match the story
- Ensure dialogue sounds natural
- Check for mixed scripts/encoding issues

4. Manual Review Checklist
- No foreign characters mixed in
- Consistent character names throughout
- Natural dialogue flow
- Varied sentence structure
