# Myanmar Translation Style Guide for "My Story"

## 1. General Tone
- **Narrative:** Use formal, literary Myanmar (စာပေဟန်). Avoid colloquial slang unless specified.
- **Dialogue:** Natural spoken Myanmar. Match the character's status and relationship.
- **Pacing:** Keep sentences concise. Chinese sentences can be long; break them into shorter Myanmar sentences for readability.

## 2. Pronoun Usage (Critical for Myanmar)
- **I/Me:** 
  - Male protagonist to elders: "ကျွန်တော်"
  - Male protagonist to peers/friends: "ငါ" or "မင်း" (context dependent)
  - Female protagonist: "ကျွန်မ" or "ငါ"
- **You:**
  - To elders/superiors: "ရှင်" or "ခင်ဗျား"
  - To peers: "မင်း"
  - To inferiors/enemies: "နင်" or "ကောင်"

## 3. Specific Terminology Rules
- **Cultivation Terms:**
  - Qi -> ချီ (Always transliterate, never translate as 'air')
  - Dantian -> ဒန်တျန် (Transliterate)
  - Meridians -> သွေးကြောမျဉ်းများ (Translate descriptively if no standard term)
- **Honorifics:**
  - Master/Shifu -> ဆရာကြီး / ဂိုဏ်းခေါင်းဆောင်
  - Junior Brother/Sister -> ညီလေး / ညီမလေး (Use based on age/rank in sect)

## 4. Formatting Rules
- Preserve all Markdown headers (#, ##).
- Keep bold (**text**) and italic (*text*) formatting.
- Do not translate proper nouns inside code blocks or specific names unless in glossary.

## 5. Cultural Adaptation
- Idioms: Translate the *meaning*, not the literal words.
  - Example: "Draw a snake and add feet" -> "အလကား အလုပ်လုပ်သည်" (Do unnecessary work)
